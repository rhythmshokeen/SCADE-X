What does my file need to look like?

Your file should have one row per step per purchase order. The minimum three columns are:

ColumnWhat it meansExample value
case_idPurchase order number — same for all steps of one POPO-2024-00123
activityThe name of the step that happenedManager Approval
timestampWhen it happened2024-03-15 09:42:00

Optional columns that make detection more accurate:

ColumnWhat it helps detect
userSegregation of duties violations (wrong person doing a step)
amountInvoice amount drift and duplicate payments
supplier_idSuppliers involved in multiple anomalous orders

Column names don't need to match exactly — SCADE will suggest mappings automatically below. If your ERP uses different activity names, configure them in Activity Setup first.
